## 프로젝트 소개

부경대학교 학생들을 위한 동아리 플랫폼, 모아동입니다.

🎓 **학생** - 동아리를 검색하고, 정보를 확인하고, 바로 지원할 수 있어요

🎪 **동아리 운영진** - 동아리 정보를 수정하고, 지원서를 관리할 수 있어요

## 이런 문제가 있었어요

### 👨‍🎓 학생들은
- 중앙 동아리만 70개가 넘는데 에타에 뒤죽박죽 올라와요
- 동아리 이름을 모르면 검색도 못해요
- 익명글이랑 홍보글이 섞여서 찾기 너무 힘들어요

### 🎪 동아리는  
- 에타 말고는 제대로 홍보할 곳이 없어요
- 한번 올린 모집 정보를 수정할 수 없어요

<br />


## 이렇게 해결했어요

### 📂 분과별 카테고리
체육, 문화, 학술... 원하는 분야를 바로 찾을 수 있어요

### ⚙️ 관리자 페이지
동아리 담당자가 모집 기간과 모집글을 직접 수정할 수 있어요

### 🖱️ 원클릭 지원
지원하기 버튼으로 구글폼, 네이버폼, 모아동 자체의 지원 폼에 바로 연결돼요

### 🎉 정보부터 지원까지 한 곳에서
더 이상 에브리타임을 뒤질 필요 없어요. 여기서 다 해결돼요

<br />


## 이렇게 발전하고 있어요

### 🤝 총동아리연합회와 함께
학교 산하 기관인 총동아리연합회와 협업하며, 전교생이 더 쉽게 동아리를 찾고 참여할 수 있도록 서비스를 개선하고 있어요.
- 각 동아리의 행사 및 일정 정보 제공

- 동아리 소개 한마당 등 교내 행사와 연계


### 💬 지속적인 피드백 수렴
동아리 운영진과 학생들의 실제 사용 경험을 바탕으로, 의견을 듣고 빠르게 반영하며 서비스 전반을 지속적으로 개선하고 있어요.

- 동아리 운영진들과의 주기적인 미팅

- 학생 피드백을 바탕으로 한 기능 개선
