장소 선택

장소명 입력과 좌표 선택은 분리되어 있습니다. 아래 검색창이나 지도를 사용해 위도/경도를 선택하세요.

카카오 지도 JavaScript 키가 서버에 설정되어야 합니다

`application.properties`의 `dev.portal.kakao-javascript-key` 또는 환경변수로 키를 설정한 뒤 페이지를 새로고침하세요.

게시글을 선택하거나 새 게시글 작성 모드로 전환하면 위치를 선택할 수 있습니다.
주소를 검색하면 후보가 여기에 표시됩니다.
카카오 지도 JavaScript 키를 저장하면 실제 지도가 이 영역에 표시됩니다.
이미지 URL을 입력하거나 업로드하면 여기에서 미리볼 수 있습니다.

기존 게시글의 `clubId`는 읽기 전용입니다. 저장 후 목록을 다시 조회해 서버 기준 최신값으로 동기화합니다.
